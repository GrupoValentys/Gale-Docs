1.- Instalando las Dependencias para el Ambiente de Desarrollo:

(Omitir este paso si ya has configurado previamente un ambiente de desarrollo para otro proyecto).
Antes de comenzar a configurar el ambiente de desarrollo son necesarias las siguientes dependencias (En el orden presentado):

  • Git: Control de versiones para nuestro proyecto... solo con esa descripción basta...
  • Microsoft .NET 4.5 o vNext (Linux o OSX): Framework de Trabajo .NET y dependencia principal , de nuestros proyectos
  • SQL Server 2008: Uno de los motores de base de datos, que puede utilizar la API RESTful.
  • IDE para desarrollo: Puede ser Xamarin Studio o Visual Studio 2013 o Superior, para poder codificar de forma simple nuestra API.


2.- Configurando el Ambiente de Desarrollo:

Una vez instaladas las dependencias bases para nuestra API, podemos configurar un ambiente de desarrollo para nuestro proyecto.

Comenzaremos descargando nuestra plantilla base para el desarrollo: # Hacemos una copia simple del repositorio (bare clone) git clone https://github.com/dmunozgaete/Gale-API-Seed-Project.git MyFirstAPI # Abrimos la solución con nuestro IDE favorito ;)! start API.sln
